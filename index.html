About Me

I'm a self-taught web developer that start to code in late 2018. Usually, I prefer to code from scratch and use a framework as less as possible, though I still like to tinker with Bootstrap-4, Bulma.io and Materialize for style.

I also try as best as I can to always implementing the DRY approach in every project that I work with. Below are list of tech I use beside the standart HTML, CSS and JavaScript.

  • Python as backend with Flask
  • NodeJS
  • SASS as CSS pre-compiler
  • React for modularity
  • Bootstrap-4, Bulma.io and Materialize as CSS-frameworks
  • PostgreSQL, MariaDB/MySQL as SQL Database
  • MongoDB as NoSQL Databse
  • Git + GitHub for version controller
